# 🌾 Al Madina Traders

A modern, fully responsive agricultural business website built using **HTML, CSS, and JavaScript** for **Al Madina Traders** — a trusted spray and agricultural products dealership.

## ✨ Features

- 🌗 Light / Dark Mode Toggle
- 🎞️ Auto Sliding Hero Carousel
- 📱 Fully Responsive Design
- 🛒 Product Showcase Section
- 🏷️ Brand Partnership Section
- 📊 Animated Counters
- ⚡ Smooth Scroll & Reveal Animations
- 💬 WhatsApp Integrated Contact Form
- 📞 Direct Call & WhatsApp Buttons
- 🧭 Sticky Navbar with Mobile Menu
- 🔻 Large Modern Footer
